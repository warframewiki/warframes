# Trinity

Keep 3rd ability active for extra survivability, use 4th ability to heal when needed.\
Viral priming -> subsumed 1st ability on enemy -> 2nd ability on same enemy.

* Steel Path

{% hint style="info" %}
You should target enemies with a lot of health.
{% endhint %}
